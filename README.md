# Three.js 生日照片爱心

静态 Three.js 网站，使用浏览器 ES Modules 加载 `three@0.164.1` 及官方后期处理模块。

## 三维爱心

- 爱心使用隐式 XYZ 心形方程进行采样：`sqrt(x² + z²)` 与 `y` 同时决定表面和内部位置，不是二维轮廓叠加随机深度。
- 外层 760 个红、粉、金色粒子勾勒轮廓；内部 230 个稀疏粒子补充体积。
- 因为 X 和 Z 使用同一径向方程，绕 Y 轴从 0°、45°、90°、135° 观察仍保留双心瓣、中央凹槽和底部尖角。
- 按 `T` 键显示包围盒，并自动环绕相机 360° 检查爱心体积；再次按 `T` 退出。
- 红色外轮廓、心瓣、凹槽和尖角采用独立加权粒子层；金色中层补充体积，暗色内部保持稀疏。
- 爱心以缓慢心跳节奏在 `1.0 → 1.03 → 1.0` 间呼吸，粒子与照片随主体轻微扩散和收缩。

## 照片与交互

- 使用 9 张真实照片，对应 `photos/photo-01.jpg` 至 `photos/photo-09.jpg`。
- 每张照片拥有固定的 `floatPosition`、`heartPosition` 与 `focusPosition`，不使用随机照片位置。
- `floating` 为固定漂浮入场状态；开启礼物后切换至 `heart`；点击照片切换至 `focus`。
- 照片固定在心瓣、中心、侧面、底部和前后表面附近，并轻微朝向镜头以避免旋转时不可见。
- 点击照片可抽出并放大查看；点击空白处返回全景。
- 每张照片都有正反两个纹理平面与半透明暖白玻璃细边，不会显示纯色背面。

## 星尘与卡片质感

- 远景使用 1,450 个小型星尘；中景使用 680 个红金星云粒子；近景保留 58 个大颗粒 Bloom 光点，均使用 `BufferGeometry`。
- 照片边框为半透明暖白玻璃材质，带极轻金色发光与投影层，不会遮挡照片内容。
