# Three.js 生日照片爱心

静态 Three.js 网站，通过浏览器 ES Modules 加载 `three@0.164.1` 和官方后期处理模块。

## 核心交互

- 严格使用 9 张真实照片，分别对应 `photos/photo-01.jpg` 到 `photos/photo-09.jpg`。
- `heartPositions[]` 定义远景下的固定三维爱心结构；照片不会随机散落。
- `focusPositions[]` 定义单张照片查看位置。点击照片后，它会移动到屏幕中心并按屏幕尺寸放大；其余照片留在爱心中并降低透明度。
- 照片由正反两个真实纹理 `Plane Mesh` 组成，保持原图宽高比例；两面均为照片内容。

## 视觉层次

- 1,600 个红/粉/金色粒子勾勒爱心轮廓，650 个稀疏星尘构成远景宇宙背景。
- 爱心具有轻微自转与呼吸动画，但粒子亮度不会盖住照片主体。
- 使用低强度 Bloom 和轻度景深；聚焦照片时背景进一步虚化。

## 部署

同时上传 `index.html`、`photos/` 和 `virtual.flac`。访问者网络还需要能够加载 `unpkg.com` 的 Three.js 模块。
